# 🚗 Car Price Predictor

This is a Machine Learning web app built using Streamlit.

## 📌 Features
- Predicts car price based on:
  - Car Name
  - Company
  - Year
  - KMs Driven
  - Fuel Type

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 🚀 How to Run
pip install -r requirements.txt
streamlit run app.py

## 📊 Machine Learning Model
Linear Regression model trained on cleaned_car.csv dataset.
